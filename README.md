## 🛠️ Setup
yarnかnpmでお願いします!

```bash
npm install
# or
yarn install
```

## :sparkling_heart: Getting Started
### :dizzy: 実行
以下のコマンドで、開発サーバーを実行します

```bash
npm run dev
# or
yarn dev
```

[http://localhost:3000](http://localhost:3000)を開いて、結果を確認します。

### :hand: 終了
以下のコマンドで、開発サーバーを終了します

```shell
ctl + c
```

バックエンドが落ちているかもしれません！
